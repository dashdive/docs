---
title: 'S3'
---


**S3 Event Notification**: This endpoint receives notifications for events happening in an S3 bucket, like file uploads.


## Request Body

| Headers      |                  |
| ------------ | ---------------- |
| content-type | application/json |

```json title="Example request"
{
  "action": "ListBuckets",
  "timestamp": "2019-08-24T14:15:22Z",
  "provider": "aws",
  "customerId": "string",
  "featureId": "string",
  "clientType": "string",
  "clientId": "string"
}
```

## Code Snippets

```shell title="Shell + Curl"
curl --request POST \
  --url http://ingest.dashdive.com/s3 \
  --header 'content-type: application/json' \
  --data '{"action":"ListBuckets","timestamp":"2019-08-24T14:15:22Z","provider":"aws","customerId":"string","featureId":"string","clientType":"string","clientId":"string"}'
```

## Responses

### 200

Event processed successfully

| Headers      |                  |
| ------------ | ---------------- |
| content-type | application/json |

**Example `response` for `application/json`**

```json
{
  "message": "string"
}
```

### 400

Invalid event data

### 500

Server error
