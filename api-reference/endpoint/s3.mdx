title: "S3"
description: "Object Storage Event"
---

This endpoint receives object storage events, like a file upload to S3, and logs it.

| Headers      |                  |
| ------------ | ---------------- |
| Content-Type | application/json |
| X-API-Key    | \<x-api-key\>    |

<RequestExample>

```shell cURL
curl --request POST \
  --url https://ingest.dashdive.com/s3 \
  --header 'Content-Type: application/json' \
  --header 'X-API-Key: <x-api-key>' \
  --data '{
  "action": "ListBuckets",
  "timestamp": "yyyy-MM-ddTHH:mm:ssZ",
  "provider": "aws",
  "customerId": "string",
  "featureId": "string",
  "clientType": "string",
  "clientId": "string"
}'
```

```python Python
import requests
```

```go Go
package main

import (
	"fmt"
	"strings"
	"net/http"
	"io/ioutil"
)

func main() {

	url := "https://ingest.dashdive.com/s3"

	payload := strings.NewReader("{\n  \"action\": \"ListBuckets\",\n  \"timestamp\": \"yyyy-MM-ddTHH:mm:ssZ\",\n  \"provider\": \"aws\",\n  \"customerId\": \"string\",\n  \"featureId\": \"string\",\n  \"clientType\": \"string\",\n  \"clientId\": \"string\"\n}")

	req, _ := http.NewRequest("POST", url, payload)

	req.Header.Add("X-API-Key", "<x-api-key>")
	req.Header.Add("Content-Type", "application/json")

	res, _ := http.DefaultClient.Do(req)

	defer res.Body.Close()
	body, _ := ioutil.ReadAll(res.Body)

	fmt.Println(res)
	fmt.Println(string(body))
}
```

# Body Options

The S3 endpoint takes a POST request with a body of one of the thirty eight different event types below

## List Buckets Event

<ResponseField name="ListBuckets" type="">
  <Expandable title="Properties">

      The type of action, which is 'ListBuckets'.
    </ResponseField>
    <ResponseField name="timestamp" type="string">
      The date and time when the action was performed, in the ISO 8601 date-time format (yyyy-MM-ddTHH:mm:ssZ), representing UTC time.

    </ResponseField>
